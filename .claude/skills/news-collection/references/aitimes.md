# aitimes (AI타임스) 수집 가이드

국내 대표 AI 전문 매체. 원문이 이미 한글이므로 번역은 불필요하고 **요약·정제**에 집중한다.

## 수집 경로
- 메인: `https://www.aitimes.com`
- 최신기사 목록을 우선 확인. WebFetch로 메인/섹션 페이지를 가져와 최신 기사 링크를 추출한다.
- 보조: WebSearch `site:aitimes.com` + 최근 키워드(예: 모델 출시, 투자, 정책)로 최신순 확인.

## 선별
- 오늘~어제 게재 기사 우선.
- 단순 해외 토픽 재탕보다 분석·국내 동향·독점 보도 우선.
- 광고/협찬성(보도자료 그대로) 기사는 후순위.

## 이미지
- aitimes 기사는 대부분 대표 이미지가 있다. 기사 페이지 `og:image` 추출.

## 작성
- `title_ko` = 기사 제목(필요 시 군더더기만 정리). `title_orig` 동일.
- `summary_ko` = 기사 핵심 3~5문장. 기자 논평보다 사실 중심.
